== *bitcask.open*(_String_) -> _Userdata_

=== Arguments
[options="header",width="72%"]
|===
|Type |Description
|string |Path for directory hierarchy
|===

=== Returns
[options="header",width="72%"]
|===
|Type |Description
|userdata |Database lock
|===
